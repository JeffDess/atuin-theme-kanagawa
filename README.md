# Kanagawa Atuin Theme

A simple colorscheme for Atuin.

![Screenshot](screenshot.png)

## Install

- Copy kanagawa.toml in your `~/.config/atuin/themes` (or in `$ATUIN_THEME_DIR`
  if set).

- Add these lines to `~/.config/atuin/config.toml`

  ```toml
  [theme]
  name = "kanagawa"
  ```
